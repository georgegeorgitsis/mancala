## Getting started

### Introduction
This bol.com game is a terminal board game. 

### Installation

#### How to run

```
$ git clone https://gitlab.com/bolcom/georgegeorgitsis
$ cd georgegeorgitsis
$ mvn compile
$ mvn exec:java -Dexec -Dspring.main.banner-mode=off
```

#### How to test
```
$ mvn test
```
